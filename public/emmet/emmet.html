Emmet Cheat Sheet

Create an unordered list with three empty elements:

ul>li*3

> indicates the following element is a child

Create this whole entry with the above example

article.intro>p+section.sample>section.explanation+section.output>ul>li*3
an article with class="intro" containing a paragraph, then a section with class="sample" containing a section with class="explanation", then another section with class="output" containing the unordered list described above

Create whole example above with Lorem Ipsum text of 10 words each for each list item - no explanation in this case

article.intro>p+section.sample>section.output>ul>li*3>lorem10
  • Lorem ipsum dolor sit, amet consectetur adipisicing elit. Quis, obcaecati.
  • Quasi laboriosam inventore reiciendis quos, optio voluptatibus quis provident aut.
  • Nobis sint quidem minus doloribus cumque ipsam dolor perspiciatis! Libero.

Add a link to each list item above with 3 words of Lorem Ipsum text

article.intro>p+section.sample>section.output>ul>li*3>a:link>lorem3
